# แอปพลิเคชันคุณค่าทางโภชนาการของเด็กนักเรียน (อายุ 6-11 ปี)

แอปพลิเคชันให้ความรู้เกี่ยวกับโภชนาการที่เหมาะสมสำหรับเด็กวัยเรียนอายุ 6 ถึง 11 ปี โดยเน้นการให้ข้อมูลที่เข้าใจง่าย ทั้งในเรื่องของปริมาณพลังงานที่ร่างกายต้องการในแต่ละวัน และสัดส่วนของอาหารทั้ง 5 หมู่ตามหลักธงโภชนาการ 

## คุณสมบัติหลัก (Features)
- **ภาพรวมพลังงาน (Energy Overview):** แสดงข้อมูลแคลอรี่ที่เด็กวัย 6-11 ปีต้องการต่อวัน (1,400 - 1,500 kcal)
- **หมวดหมู่อาหาร (Food Groups):** แสดงข้อมูลสัดส่วนอาหารที่แนะนำต่อวันแบ่งตามหมวดหมู่ ได้แก่ ข้าว-แป้ง, เนื้อสัตว์, ผัก, ผลไม้, นม, และไขมัน/น้ำตาล
- **หน้ารายละเอียด (Detail Screens):** เจาะลึกข้อมูลของอาหารแต่ละหมวดหมู่ ปริมาณที่แนะนำ และข้อแนะนำในการบริโภค
- **ข้อแนะนำสุขภาพ (Health Tips):** คำแนะนำเพิ่มเติมเกี่ยวกับการดื่มน้ำ การออกกำลังกาย และการพักผ่อน

## เทคโนโลยีที่ใช้ (Tech Stack)
- Flutter Framework (รองรับ Android, iOS, Web และ Desktop)
- Material 3 Design
- ภาษา Dart

## การติดตั้งและการใช้งาน (Setup & Run)
1. ตรวจสอบว่ามีการติดตั้ง Flutter SDK ในเครื่องของคุณแล้ว
2. ทำการติดตั้ง Dependencies ด้วยคำสั่ง:
   ```bash
   flutter pub get
   ```
3. รันแอปพลิเคชันด้วยคำสั่ง:
   ```bash
   flutter run
   ```
